## 💻 HOW TO RUN THE PROJECT

### View the Dashboard (Recommended First Step)
```bash
# Option 1: Open directly in browser
# Double-click: report/index.html

# Option 2: Open from terminal (Windows)
start report/index.html

# Option 3: Open from terminal (Mac)
open report/index.html

# Option 4: Open from terminal (Linux)
xdg-open report/index.html
```

### Run Interactive Web App
```bash
streamlit run app.py
```
Then visit http://localhost:8501

### Train Model
```bash
python src/train.py
```

### Generate Visualizations
```bash
python generate_report_data.py
```

### Make Predictions
```bash
python src/predict.py
```
